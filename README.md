# Metal Chord Progression Builder

A React-based tool for creating chord progressions for melodic death metal.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## Features

- Select root note and scale type (Major, Natural Minor, Harmonic Minor, etc.)
- View harmonized scale with chord functions
- Build chord progressions with customizable bars and time signatures
- Play back progressions with audio synthesis
- Copy progressions to clipboard
- Support for various time signatures and BPM settings

## Build

To build for production:
```bash
npm run build
```
